# HorseSystem000  
三連複フェアオッズ × 複勝ヘッジ戦略の数理モデル実装

本プロジェクトは、  
**三連複のフェアオッズ計算** と **複勝ヘッジ** を組み合わせた  
独自の競馬投資ロジックを Python で実装したものです。

- 三連複の「勝負ゾーン」を数学的に定義  
- フェアオッズから「買うべき最大オッズ M」を決定  
- 複勝ヘッジにより損失ゼロ条件を満たす  
- p_t（的中確率）を q_i から構成  
- 人気帯フィルタにより対象馬を最適化  

競馬の勘や経験ではなく、  
**確率モデルとオッズ構造に基づく再現性のある戦略** を目指しています。

## 🧠 ロジック概要

### 1. 複勝オッズから 3着以内確率 q_i を推定  
複勝オッズ F_i から、以下の式で 3着以内確率 q_i を計算します。



\[
q_i = \frac{1/F_i}{\sum_j 1/F_j}
\]



これは「複勝オッズの逆数を正規化したもの」で、  
**三連複の確率構造を作る基礎** になります。

---

### 2. 三連複のフェアオッズを計算  
3頭 (i, j, k) の組み合わせについて、



\[
P(i,j,k) = q_i \cdot q_j \cdot q_k
\]




\[
O_{ijk}^{fair} = \frac{1}{P(i,j,k)}
\]



これにより、  
**理論上の三連複オッズ（フェアオッズ）** を全組み合わせで算出します。

---

### 3. 勝負ゾーンの定義  
フェアオッズを小さい順に並べ、  
**合成オッズ O(M)** が複勝ヘッジ条件を満たす範囲を「勝負ゾーン」と定義します。

複勝ヘッジ馬 A のオッズを a とすると、  
損失ゼロ条件は：



\[
O(M) > \frac{a}{a-1}
\]



---

### 4. 最大オッズ M の決定  
勝負ゾーンに含めるフェアオッズを増やしていき、  
上記条件を満たすギリギリの値を **最大オッズ M** とします。
